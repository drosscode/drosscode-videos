On *nix, get the process id with [ps][1]
or if you know the process name, do

    ps aux | grep processname

And to run any program as a daemon, use [nohup][2]


  [1]: http://www.computerhope.com/unix/ups.htm
  [2]: http://en.wikipedia.org/wiki/Nohup
1. To get the running process&#39; identifier, use the `getpid()` function.
2. To create a daemon, i.e. a detached process running in the background, follow [these instructions][1].

  [1]: http://www-theorie.physik.unizh.ch/~dpotter/howto/daemonize
